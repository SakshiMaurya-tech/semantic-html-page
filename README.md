# Navigation Bar Project

## Overview

This project is a horizontal navigation bar created using HTML and CSS.

## Features

 Logo on the left side
 Four navigation links in the center
 Sign-up button on the right side
 Dropdown menu under the More link
 Hover effects on navigation links
 Focus-visible styles for keyboard accessibility
 Fixed navigation bar at the top of the page

## Files

 nav.html – contains the HTML structure
 nav.css – contains the CSS styling
